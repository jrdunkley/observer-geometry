# Provenance

This bundle contains a tiny manually reconstructed score table from the public Hugging Face Open LLM Leaderboard page for three Yi-family models and four tasks:

- IFEval
- BBH
- MATH Lvl 5
- MMLU-PRO

Only the numeric facts in `raw/score_slice.csv` are redistributed here. No raw leaderboard artifact or large dataset is bundled.
