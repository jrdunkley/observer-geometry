# Provenance

Source counts were transcribed from:

- A. E. Jones et al., *The strong no-signalling inequalities of quantum mechanics are not violated experimentally*, SciPost Physics 10, 017 (2021)
- SciPost PDF URL: `https://scipost.org/SciPostPhys.10.1.017/pdf`

The transcribed values are the four-context coincidence counts listed in Table A2 / Appendix C, together with the setting labels shown immediately below that table.

This bundle keeps two layers separate:

- exact extracted counts and setting labels
- encoded Gaussian-surrogate covariance matrices derived from those counts for downstream `nomodescent`
