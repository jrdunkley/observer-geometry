# Benchmark Blindness Encoding

This example is explicitly synthetic.

- benchmark task inclusion and benchmark scores are encoded as exact tables
- observer matrices over latent capability directions are encoded as a finite
  modelling family
- the first output is underdetermination, not a fake clean answer
- only after an explicit observer-family choice does `nomodescent` receive a
  problem spec

The claim is only that the evidence layer keeps benchmark facts separate from
capability modelling choices.
