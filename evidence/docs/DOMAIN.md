# Domain

`evidence v0.3` supports a narrow evidence-encoding regime:

- explicit numeric matrices
- structured numeric tables
- finite protocol descriptions
- finite observer hypothesis families
- small curated text claims and notes

It does not support:

- arbitrary PDF ingestion
- OCR-heavy extraction
- free-form literature summarisation
- broad autonomous web search
- unconstrained semantic reconstruction of latent models

The package is useful because it refuses to pretend those deferred tasks are
already solved.
