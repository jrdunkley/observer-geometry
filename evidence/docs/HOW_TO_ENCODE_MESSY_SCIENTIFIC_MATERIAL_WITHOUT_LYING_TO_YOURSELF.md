# How To Encode Messy Scientific Material Without Lying To Yourself

- Exact extraction is not the same as convenient interpretation.
- If you had to choose one of several observer families, that choice is an
  encoded inference, not an exact fact.
- If the source leaves several encodings open, preserve the ambiguity.
- A refusal is better than a false clean encoding.
- Separate source facts from modelling notes.
- Keep authoritative downstream inputs narrow.
- Treat `underdetermined_evidence` as a valid scientific output.
- Use the ambiguity-collapse route to decide what evidence would help next.
