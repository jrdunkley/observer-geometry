# Release Note

`nomodescent v0.2` is the first exact scientific workbench layer above the
frozen `nomogeo v0.1.0` kernel.

## What It Is

- a validated problem contract for observer-geometry problems
- an exact descent engine in the linear / Gaussian regime
- an obstruction-certificate surface
- an audit-first output layer
- three worked examples:
  - Bell as compatibility / descent
  - free Gaussian RG as repeated quotient descent
  - replication fragility as observer mismatch
- a narrow operational Quotient Descent core:
  - factorisation
  - common refinement
  - common descent
  - false collapse diagnostics
  - finite-family refinement search
- three external comparison packs:
  - Bell versus correlator-level CHSH / arcsine logic
  - protocol mismatch versus canonical correlation analysis
  - free Gaussian RG versus Schur-complement elimination

## What It Is Not

- not a generic heuristic science engine
- not an automatic PDF parser
- not a nonlinear inference framework
- not a broad search layer over arbitrary observer spaces

## Boundary

The exact engine is theorem-grade only inside the matrix-based linear / Gaussian
regime. A small audited approximate PSD search is included, but it is always
labelled as approximate.
