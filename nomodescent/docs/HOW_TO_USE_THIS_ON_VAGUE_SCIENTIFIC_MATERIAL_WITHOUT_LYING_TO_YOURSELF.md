# How To Use This On Vague Scientific Material Without Lying To Yourself

## Separate Facts From Guesses

Encode exact constraints separately from approximate assumptions. If a protocol
really gives a covariance estimate, enter it as evidence. If you are inferring a
latent dimension or identifying two observers by analogy, put that in the
assumption ledger instead.

## Do Not Flatten Distinct Observer Families

Two studies can probe the same latent object with different observers. Do not
force them into one observer just because the labels sound similar. Put the
observer matrices in explicitly and let the relation test decide whether one
factors through the other.

## Use “Underdetermined” Honestly

If the exact engine cannot decide compatibility because the affine completion
family is unresolved, `underdetermined` is the right scientific output. It is
not a failure mode. It is information about what the current evidence does not
pin down.

## Treat Approximate Layers As Approximate

If you enable the low-dimensional PSD-feasibility search, the audit layer will
mark the result as approximate. Do not quietly report that as theorem-grade.

## Read The Audit Before Reading The Story

Every result tells you:

- which assumptions were exact
- which assumptions were approximate
- which inputs were treated as authoritative
- which theorem layer the conclusion belongs to
- which falsification route would break the current conclusion fastest

That is the discipline. Read it first.
